# Yet Another CHIP-8 Emulator (YACE)

My take on the ubiquitous CHIP-8 emulator in C as both an exercise in C and learning about emulators.

# Build instructions
```
cd ./buid/
cmake
make 
make run
```

The code should run the Tetris ROM in ./roms/
